>btc [txid/address (optional)]
  • Group: Crypto
  • Description: This command has three main functionalities.\n\n**1. General BTC Information** - Just using the command >btc will give you some general information about the crypto Bitcoin. An example of the information give is price and the change in 24 hours.\n\n**2. Address Search** - If you supply an address with the command, >btc `address`, the bot will give you a summary of the bitcoin wallet. For example, the current value of the wallet in BTC and USD.\n\n**3. Transaction ID Search** - If you supply a transaction ID with the command, >btx `txid` (txid = Transaction ID)`, the bot will give you a summary of the transaction, for example the amount of BTC transacted.
  • Required Permissions: None
