Utility

These are the utility commands in Carnage Bot.

Command Description Permissions Required
>echo message Repeats a message you give it. None
>commandlist Sends you a dm with all available commands. None
>define word Defines a word. None
>suggest suggestion Sends your suggestion to the devs None
>hosting Gives you information about the current hosting. None
